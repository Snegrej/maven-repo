Class file is located in:
src/main/java/com/mycompany/app/

Resource data files located in:
src/main/resources/txtFiles/

Jar file is located in:
target/

- With java, to execute the jar file from pom.xml run:
  java -cp target/my-app-1.0-SNAPSHOT.jar com.mycompany.app.App

- With maven, to compile a new jar from pom.xml run:
  mvn install
